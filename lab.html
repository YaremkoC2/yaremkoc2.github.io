What is the Lab?

The Lab is a space for experiments. Small, interactive, or unfinished ideas that I'm playing with. Unlike the main Projects section, these aren't polished or fully built out, but some of these may grow into full projects.

DOM Tetris

A Tetris clone built directly in the DOM with HTML, CSS, and JavaScript. It's a fun way to explore rendering logic, collision detection, and game loop timing. It's currently in a somewhat functional state, but still needs some polish. No mobile support yet, but it works well on desktop browsers.
